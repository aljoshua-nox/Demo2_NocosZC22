# Django Project - Demo_NocosZC22 
 
## Description 
This is a Django practice project from Lecture 9: Front-End Architecture: Logic and Data Models
 
## Features 
- Django project setup 
- Members app with views and templates 
- SQLite database with Member model 
 
## How to Run 
 
1. Clone the repository 
2. Create virtual environment: `python -m venv venv` 
3. Activate virtual environment 
4. Install dependencies: `pip install django` 
5. Run migrations: `py manage.py migrate` 
6. Run server: `py manage.py runserver` 
7. Visit: http://127.0.0.1:8000/
